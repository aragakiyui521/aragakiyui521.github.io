## 无可奉告！


【满江红 · 长者】

看香江无良，岂言天命俱已亡。
十三亿，济济一心，处处严防。
黑白无常肝胆裂，阎罗殿上续命忙。
要阴阳二界识长者，裤腰长。
千万心，为痴狂。
一秒到，仰天望。
吾等力微薄，但求上苍。
昔年谈笑沪宁港，八宝山上能安葬？
待红衣背篓归来日，还图样。
